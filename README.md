# Ejercicio de generación de codigo intermedio del curso de Organización de Lenguajes y Compiladores 2
 Este proyecto es un ejercicio para generar codigo 3 direcciones para expresiones logicas, relacionales y aritméticas. 
 Desarrollado por Ronald Gabriel Romero González - 201701048.
 
 En el archivo entrada.txt se recibe una expresión y en el archivo salida.txt se escribe el codigo 3 direcciones resultante. 
 Para ejecutarlo se debe abrir una consola y escribir: python .\gramatica.py
